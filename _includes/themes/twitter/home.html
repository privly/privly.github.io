Take Control

With Privly you can use any website without giving that website access to your content. You control your digital life. Learn more about controlling your digital life with Privly.

Download

Privly changes the way your web browser works with a "browser extension". Learn more about the Privly extensions on the download page.

Get Hosting

Anyone can host a server for Privly content, but we maintain a list of hosts for you if you don't have the required expertise. Learn about the Privly hosts.

Get Involved

Interested in the Project and want to contribute development for the privly project? Check out privly.org for more information on the privly project tools and how to get setup developing.

Flexible

Privly is a flexible framework, meaning web developers can build applications to keep Privly fresh and powerful. Learn More about Privly's design philosophy

Open Source

Privly is developed under the MIT license and is available on GitHub. The best place to get plugged into Privly development is on IRC.

Non-Profit

The Privly project is governed by the Privly Foundation, which is a US non-profit (pending IRS non-profit determination). Learn more about the foundation.
