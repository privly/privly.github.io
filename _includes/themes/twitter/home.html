Take Control

With Privly you can use any website without giving that website access to your content. You control your digital life. Learn more about controlling your digital life with Privly.

Download and Get Hosting

Privly changes the way your web browser works with a "browser extension". Learn more about the Privly extensions on the download page. Anyone can host a server for Privly content, but we maintain a list of hosts for you if you don't have the required expertise.

Flexible

Privly is a flexible framework, meaning web developers can build applications to keep Privly fresh and powerful. Learn more about Privly's design philosophy.

Open Source Non-Profit

Privly's source code is available on GitHub. To contribute code you should read "How to Get Started" on privly.org. The project is governed by the Privly Foundation, which is a US non-profit (pending IRS non-profit determination). Learn more about the foundation.
